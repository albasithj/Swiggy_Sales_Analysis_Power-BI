# Swiggy_Sales_Analysis

## Project Overview
This project focuses on analyzing Swiggy food delivery data to understand
customer behavior, order trends, restaurant performance, and revenue patterns.
The goal is to extract meaningful insights that can support business and
operational decision-making.

## Tools & Technologies Used
- Power BI (Dashboard and Visualization)
- Python (EDA), if applicable
- Pandas and NumPy
- Excel / CSV
- Power Query
- DAX

## Dataset Information
The project uses multiple datasets related to orders, restaurants, customers,
and delivery details. Due to file size limitations, only sample datasets with
the same structure are uploaded to this repository.

## Data Processing and Analysis
- Data cleaning and preprocessing
- Handling missing values and duplicates
- Data transformation and feature understanding
- Relationship creation between datasets
- Creation of calculated measures and metrics
- Trend and performance analysis

## Key Analysis and Insights
- Order volume trends over time
- Top-performing restaurants and cuisines
- Customer ordering behavior
- Revenue and pricing analysis
- Delivery and demand patterns

## Outcome
- Gained a clear understanding of customer and order behavior.
- Identified high-performing restaurants and popular food categories.
- Observed trends in order frequency and revenue contribution.
- Built an interactive and reusable analytical model.
- Strengthened skills in data cleaning, analysis, and visualization.
